# BMS故障代码整型映射表

为了方便在ROS话题中对BMS（电池管理系统）的故障进行订阅和处理，最新代码中将原有的字符串格式重构为 `std_msgs/UInt64` (64位无符号整数) 类型的位掩码(Bitmask)输出。

每一个故障类型对应 `UInt64` 中的一个独立位（Bit）。当某位的值为 `1` 时，代表发生了对应故障。当整体数值为 `0` 时，代表系统正常。

## 解析示例代码 (C++)

如果需要在另外一个 ROS 节点中订阅这个消息，可以通过按位与(`&`)运算来检查是否有特定故障：

```cpp
void faultCallback(const std_msgs::UInt64::ConstPtr& msg) {
    if (msg->data == 0) {
        ROS_INFO("BMS 系统正常");
        return;
    }

    // 检查是否有充电低温告警 (Bit 0)
    if (msg->data & (1ULL << 0)) {
        ROS_WARN("充电低温告警");
    }
    
    // 检查是否有预充失败 (Bit 33)
    if (msg->data & (1ULL << 33)) {
        ROS_ERROR("预充失败");
    }
}
```
